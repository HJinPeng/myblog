HTTP--cookie

Posted on 2019-03-29 | In HTTP

1. 什么是cookie？

cookie是会话跟踪技术，是用在客户端的
session也是会话跟踪技术，但是是用在服务器端的

2. cookie有什么作用？

cookie可以将网页的数据保存在浏览器本地

3. cookie的生命周期

默认情况下，cookie是属于一次会话（即浏览器关闭就消失），但是可以通过设置expires = 过期时间 设置过期时间

4. 注意点

  1. cookie默认不会保存任何数据
  2. cookie不能一次性设置多条数据，要想保存多条数据，只能一条一条的设置
  3. cookie有大小限制： 4kb左右，小于4KB
  4. cookie有个数限制： 20~50

5. cookie的作用范围

同一个浏览器下的同一个路径。
如果在同一个浏览器下，路径的下级也可以访问cookie，而上级路径不可以访问到cookie，但可设置path使整个网站都能访问到cookie

如：
http://localhost:8081/php_learn/cookie.html
http://localhost:8081/php_learn/cookie2.html
是属于同一浏览器下的同一路径，路径都是http://localhost:8081/php_learn/
而http://localhost:8081/php_learn/test/index.html
也可以访问到cookie。

设置cookie路径

document.cookie = "name = 123; path:/; domain=hjinpeng.top";
设置了cookie在根目录下，根目录下的所有文件都可以访问到cookie，并且在一级域名相同情况下，二级域名也可以访问到cookie
