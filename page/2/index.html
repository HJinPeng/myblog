jquery--attr(),prop()

Posted on 2019-03-17 | In jquery

1. attr() 和 removeAttr()

  1. attr()传入一个参数，表示获取属性节点的值。无论找到多少个元素，都只返回第一个
  2. attr()传入两个参数，表示设置某属性的值。无论多少个元素，都一起设置。如果参数存在，则改变属性值，参数不存在，则新增属性和值。
  3. removeAttr() 删除多个参数用空格隔开

2. prop() 和 removeProp()

  1. prop()和attr()方法一样
  2. removeProp()和removeAttr()方法一样
  3. 重点： prop()和attr()基本一样，但是当属性返回值为true / false 时，使用prop();如selected,checked,disabled

css-透明img背景设置

Posted on 2019-03-17 | In css

1. 圆形img图片，可以设置背景颜色，边框弧度

在这里，border-radius: 50% 设置成大于50%的值都可以，理解如下：

  1. border-radius 同时设置了四个角
  2. 百分比 = 取点的地方 / 边长 ， 比方：border-top-right-radius: 70%; 是取元素的上边框的边长70%和右边框的70%的两个位置做一个弧形。
  3. 当边框是一个正方形时，border-radius: 50%;（>50%）可以形成一个圆形
  4. 当边框是一个长方形时，border-radius: 50%;（>50%）可以形成一个正椭圆形
  5. 若左右 / 上下 两个弧度百分比 相等 且 相加>100%，则会自动变成100%，两边都变成50%

css-display的inline,inline-block,block

Posted on 2019-03-16 | In css

1. display: block

  1. 设置为块级元素，独占一行
  2. 可以设置width,height,margin,padding

2 .display: inline

  1. 设置为行内元素，和其他行内元素共享一行，当一行拍不下才会换行
  2. 不可设置width,height
  3. margin-top,margin-bottom,padding-top,padding-bottom不会产生效果

3. display: inline-block

  1. 作为一个行内元素，但是具有block的属性特征，如有width,height
