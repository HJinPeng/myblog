2. 初始化颜色

  • 白色：未被访问
  • 灰色：被发现的顶点
  • 黑色：已经输出的顶点

3. 广度优先搜索

实现：

  • 创建一个队列
  • 将v标注为被发现的（灰色），并将v放入队列Q
  • 如果Q非空，执行下面的步骤：
    • 将v从Q中取出
    • 将v标注为被发现的灰色
    • 将v所有的未被访问过的邻接点（白色），加入到队列中
    • 将v标志位黑色

4. 深度优先搜索（DFS）

思路：和二叉树的先序遍历有点像，递归调用
