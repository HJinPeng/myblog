es6--函数扩展

Posted on 2019-04-13 | In ES6

1. 基本用法

ES6前不能直接为函数的参数指定默认值。

重点看最后一句，当传入y=’’时，但是y的布尔值时false，所以使用了’World’。为了避免这种情况，通常要先判断一下参数y是否被赋值。

ES6允许给参数默认赋值。

注意：
参数变量是默认声明的，所以不能用let 和 const 再次声明

使用参数默认值时，函数不能有同名参数

参数默认值不是传值，而是每次都会重新计算，也就是说，是惰性求值的。

3. 函数的length属性

指定了默认值后，函数的length属性将返回没有指定默认值的参数个数。也就是length属性失真。
如果设置了默认值的参数不是尾参数，那么length属性也不再计入后面的参数。

4. 函数参数的作用域

一旦设置了参数的默认值，函数进行声明初始化时，参数会形成一个单独的作用域（context）

因为调用函数f()时，参数形成一个单独的作用域，在这个作用域里，y = x ，的x指向这个作用域中的x，现在x等于传入的2，所以打印出2。

6. 箭头函数

注意事项：

  1. 函数体内的this对象就是定义时所在的对象，而不是使用时所在的对象。
  2. 不可以当做构造函数，也就是不可以new。因为箭头函数没有自己的this。
  3. 不可以使用arguments，该对象不存在。如果要用，可以用rest参数代替。
  4. 不可以使用yield命令，因此不能用作Generator函数。
