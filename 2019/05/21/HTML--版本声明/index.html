Html 版本声明

一、 html 5

<!DOCTYPE html>

二、 html 4.0.1

1. html 4.0.1 Strict

<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01//EN" "http://www.w3.org/TR/html4/strict.dtd">
该DTD包括html的元素和属性，但不包括展示性的和被弃用的元素。不允许框架集

2. html 4.0.1 Transitional

该DTD包括html的元素和属性，包括展示性和被弃用的元素，但不允许框架集

3. html 4.0.1 Frameset

该DTD包括html的元素和属性，包括展示性的和被弃用的元素，也包括框架集

三、 XHTML 1.0

XHTML 1.0 和 html 4.0.1 一样有三种模式，但必须以格式正确的XML来编写标记

1. XHTML 1.0 Strict

2. XHTML 1.0 Transitional

3. XHTML 1.0 Frameset

四、 XHTML 1.1

该DTD等同于XHTML1.0 Strict ，但允许添加模型
