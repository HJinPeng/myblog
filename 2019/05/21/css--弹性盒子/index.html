css--弹性盒子

Posted on 2019-05-21 | In css

1. 父元素的属性

flex 和 inline-flex

flex-direction

主轴方向

  1. row 默认，从左到右
  2. row-reverse 横向从右到左
  3. column 主轴变成从上到下
  4. column-reserse 主轴从下到上

flex-wrap

  1. nowrap 默认值，当子元素溢出父元素时不换行，会对子元素在宽度上进行删减，使得全部排进一行
  2. wrap 当子元素溢出父元素时换行
  3. wrap-reverse 反转wrap排列

justify-content

设置弹性盒子子元素在主轴方向上的对齐方式

  1. flex-start 子元素向行起始位置对齐
  2. flex-end 子元素向行结束位置对齐
  3. center 子元素向中间位置对齐
  4. space-between 子元素会平均分配到行里
  5. space-around 子元素会平局分布在行里且两端保留子元素和子元素之间间距大小的一半

align-items

设置子元素在测轴方向上的对齐方式

  1. flex-start
  2. flex-end
  3. center
  4. baseline 将文字对齐
  5. stretch 如果为设置宽高，那么将元素默认拉伸为父元素高度

align-content

用于修改flex-wrap属性的行为，类似于justify-content，但它不是对子元素的对齐，而是对各个行的对齐
当flex-wrap: wrap时，产生了几行，使用下面的属性可以设置行之间的间距即测轴方向的对齐方式

  1. flex-start
  2. flex-end
  3. center
  4. space-between
  5. space-around
  6. stretch 默认值

2. 子元素的属性

flex-grow

根据子元素设置的flex-grow的比例分配剩余的空间

flex-shrink

默认值是 1

flex-basis

设置弹性盒子伸缩基准值

  1. auto 无特定宽度，取决于其他属性值
  2. length 用长度定义宽度，不允许负值
  3. percentage 用百分比来定义宽度

超出父元素宽度会按flex-shrink来计算

flex

flex: [flex-grow][flex-shrink][flex-basis]
默认值 flex: 0 1 auto
flex: 1 等于 flex: 1 1 0
用途: 左中右布局

order

order设置子元素出现的顺序，小的在前面，大的在后面，默认值为0，可以为负数

align-self

单独定义某一个元素在测轴上的位置

  1. flex-start
  2. flex-end
  3. center
  4. auto
  5. baseline
  6. stretch
