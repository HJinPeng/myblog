css-overflow

Posted on 2019-03-21 | In css

1. overflow: hidden

遇到的情况：section设置了背景颜色但高度为auto，section中三个div都设置了float，背景颜色不显示

原因：所有div设置了float，导致section无法根据子元素撑起高度

解决方法：在父元素section中设置清除浮动section{overflow:hidden;} ,这样浮动的子元素又可以撑起高度了

ps：若父元素section设置了高度值，则浮动的元素超出范围的部分会被隐藏

2. overflow: auto

超出边界的值不显示，但显示默认样式滚动条
