git--基本指令

Posted on 2019-03-18 | In git

1. 创建版本库

  1. 初始化Git仓库

    在一个文件夹中使用git init，就可以变成git可以管理的仓库，会生成默认隐藏的.git文件夹

  2. 添加文件到Git仓库

    2.1 git add <file>，可反复使用多次，添加多个文件

    2.2 git commit -m <message> ，一次提交

2. git status 和 git diff

git status 可以查看工作区的状态，如修改后还未添加到工作区，添加后还未提交等。

git diff可以查看文件的修改内容

3. 版本回退

  1. HEAD 指向当前版本
  2. git reset --hard commit_id 可以穿梭到任意一个版本，git reset --hard^ 指上一个版本，head^^指上上个版本
  3. 使用git log 可以查看提交的历史，确定回退到哪个版本
  4. 使用git reflog 查看命令历史

4. 撤销修改

  1. 当修改了工作区的文件时，想直接丢弃工作区的修改，可以使用git checkout --file
  2. 当修改了工作区文件且已经add到了暂存区，想丢弃修改
    1. git reset HEAD <file> 回到场景1，工作区的文件会变成暂存区的文件状态
    2. git checkout --file 丢弃工作区的文件修改状态

5. 删除文件

  1. 删除版本库中的文件，git rm 用于删除一个文件，git commit -m <message>提交
  2. 误删本地文件，但文件存在版本库中，可以使用git checkout -- file 将版本库中最新的文件下载到本地
