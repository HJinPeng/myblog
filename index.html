兼容性

Posted on 2019-05-10

1. 图片

img的默认属性inline-block会使得下面有个间隙
解决方法：
将img设置为block

2. 双倍边距

当ie6及更低版本浏览器在解析浮动元素的时候，会错误的把浮动边界加倍显示
解决方法：
给浮动元素添加声明: display: inline;

3. 默认高度（ie6）

ie6及其以下版本中，部分块元素拥有默认高度（18px高度）
解决方案：

  1. 给元素添加声明：font-size:0
  2. 给元素添加声明：overflow:hidden

4. 表单元素的行高不一致

在ie6下表单元素的高度不一致，如按钮文字的高度和input中文字的高度不一致
解决方案：
给表单元素添加float:left

5. 按钮元素默认大小不一样

解决方案：

  1. 使用a标签制作按钮
  2. 在input上写按钮的样式，要把input的边框去掉
  3. 如果按钮是一张图片，直接把图片作为按钮的背景图

6. 百分比bug

在ie6及以下浏览器中解析百分比时，会按四舍五入计算，导致50%+50%>100%
解决方法：

  1. 给右边的浮动元素添加声明：clear:right

cursor: hand只有ie浏览器能识别，cursor: pointer;在ie6以上能识别
解决方案：
使用兼容更多浏览器的属性

git--忽略文件.gitignore

Posted on 2019-05-06 | In git

.gitignore内容

#后的内容为注释，会被忽略
.txt　　 #会忽略所有.txt结尾的文件
!test.txt　　 #test.txt除外，即不被忽略
/todo　　 #仅忽略项目根目录下的 TODO 文件，不包括 subdir/todo
build/ 　　 # 忽略 build/ 目录下的所有文件
doc/.txt 　　# 会忽略 doc/notes.txt 但不包括 doc/server/arch.txt

.gitignore不生效的解决方法

把某些目录或文件加入忽略规则，按照上述方法定义后发现并未生效，原因是.gitignore只能忽略那些原来没有被追踪的文件，如果某些文件已经被纳入了版本管理中，则修改.gitignore是无效的。那么解决方法就是先把本地缓存删除（改变成未被追踪状态），然后再提交：
git rm -r --cached .
git add .
git commit -m "message"

Posted on 2019-04-22 | In json

1. 安装

npm install -g json-server

2. 任意位置创建一个文件夹：JSONSERVER

键入命令
npm init

3. 输入命令

npm install json-server --save

4. 修改JSONSERVER中的package.json

修改为：

5. 在JSONSERVER下创建文件db.json

6. 键入命令

npm run json:server

7. 打开连接

localhost:3000

8. 使用

  • 获取所有用户信息：
    http://localhost:3000/users

  • 获取id为1的用户信息：
    http://localhost:3000/users/1

  • 获取公司的所有信息
    http://localhost:3000/companies

  • 获取公司id为1的信息
    http://localhost:3000/companies/1

  • 获取所有公司id为3的用户信息
    http://localhost:3000/companies/3/users

  • 根据公司名字获取公司信息
    http://localhost:3000/companies?name=Microsoft

  • 根据多个公司名字获取公司信息
    http://localhost:3000/companies?name=Microsoft&name=APPLE

  • 获取一页中只有两条数据
    http://localhost:3000/companies?_page=1&_limit=2

  • 升序asc 降序desc
    http://localhost:3000/companies?_sort=id&_order=desc

  • 获取年龄30及30以上的用户信息
    http://localhost:3000/users?age_gte=30

  • 获取年龄在30-40的用户信息
    http://localhost:3000/users?age_gte=30&age_lte=40

  • 根据搜索信息找
    http://localhost:3000/users?q=wen
